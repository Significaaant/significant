Comment faire parler ses données

Avant de plonger, il faut traduire vos données "réelles" en langage "statistique".

Guide du Traducteur (Cas Réel)

Prenons un exemple complexe : "L'évolution de la présence du topic 'Cancer du Sein' dans les articles de 1920 à 2016."

  • 1️⃣ Année (1920, 1921...) : C'est un mesure chiffrée.
    👉 Traduction : Variable Quantitative.
  • 2️⃣ Présence du Topic (12%, 5%...) : C'est un score chiffré.
    👉 Traduction : Variable Quantitative.
  • 3️⃣ La Question : "Est-ce que ça augmente/baisse avec le temps ?"
    👉 Traduction : Vous cherchez une Corrélation (Lien).

✅ Résultat Quiz : "2 Variables Quantitatives".

Le Mystère de la "Loi Normale"

Le quiz va souvent vous demander : "Vos données sont-elles Normales ?". C'est crucial.

👀 Méthode Visuelle

Faites un histogramme. Cela ressemble-t-il à une cloche symétrique ?

  • ❌ Non : Il y a des barres géantes à gauche ou à droite (Biaisé). 👉 Non-Normal
  • ✅ Oui : C'est joli et centré. 👉 Normal
🤖 Méthode "Robo-Test"

Utilisez le test de Shapiro-Wilk (voir codes plus bas).

  • Si p-value < 0.05 : 🚨 Pas Normal (Alerte !)
  • Si p-value > 0.05 : ✅ Normal (Tout va bien) 
# Test de Normalité (Shapiro-Wilk)
shapiro.test(data$MaVariable)

# Si p-value < 0.05 -> Pas Normal
# Si p-value > 0.05 -> Normal
from scipy.stats import shapiro

stat, p = shapiro(df['MaVariable'])
print(f"P-value: {p}")

# Si p < 0.05: Pas Normal

L'interprétation (Inférence)

C'est le cœur de l'analyse : vérifier si vos hypothèses sont "Statistiquement Significatives".

🚀 Test du Chi-Carré (Chi-Square)

C'est quoi ? Un test pour vérifier l'indépendance entre deux variables qualitatives.
💡 Exemple : "Y a-t-il un lien entre le Sexe (H/F) et le Tabagisme (Oui/Non) ?"
Pourquoi lui ? Car on ne peut pas calculer de moyenne sur des mots. On compare donc les fréquences observées vs attendues. 

# Créer le tableau croisé
tableau <- table(data$Variable1, data$Variable2)

# Test du Chi-2
chisq.test(tableau)
from scipy.stats import chi2_contingency
import pandas as pd

# Tableau de contingence
tab = pd.crosstab(df['Var1'], df['Var2'])
chi2, p, dof, ex = chi2_contingency(tab)
print(f"P-value: {p}")
// Chemin
Analyses > Fréquences > Échantillons indépendants

// Options
1. Lignes : Variable 1
2. Colonnes : Variable 2
3. Statistiques : Chi-carré (coché par défaut)

ANCOVA (Analyse de Covariance)

Pourquoi ? Comparer des groupes en "effaçant" l'influence d'une variable parasite.
💡 Exemple : "Comparer l'efficacité de 3 méthodes d'enseignement, mais en contrôlant le QI initial des élèves."
Pourquoi lui ? Si tes groupes ne sont pas égaux au départ (ex: QI différent), l'ANCOVA corrige le biais mathématiquement. 

# ANCOVA (Modele linéaire)
# Y = VD, Groupe = VI, Covar = Parasite
modele <- aov(Y ~ Groupe + Covariable, data=df)
summary(modele)
import statsmodels.formula.api as smf

# fit OLS model
model = smf.ols('Y ~ C(Groupe) + Covariable', data=df).fit()
print(model.summary())
// Chemin
Analyses > ANOVA > ANCOVA

// Options
1. Variable dépendante : Y
2. Facteurs fixes : Groupe
3. Covariables : Covariable (ex: Âge, QI)

MANOVA (Multivariée)

Pourquoi ? Tester plusieurs résultats (VDs) en même temps.
💡 Exemple : "L'impact d'un médicament sur (Tension ET Cholestérol ET Poids)."
Pourquoi lui ? Évite de faire 3 tests séparés (ce qui augmente le risque Pourquoi lui ? Évite de faire 3 tests séparés (ce qui augmente le risque d'erreur) et détecte des profils globaux. 

# MANOVA
# cbind regroupe tes VDs
modele <- manova(cbind(Y1, Y2, Y3) ~ Groupe, data=df)
summary(modele, test="Wilks")
from statsmodels.multivariate.manova import MANOVA

# Formule style R
ma = MANOVA.from_formula('Y1 + Y2 + Y3 ~ Groupe', data=df)
print(ma.mv_test())
// Chemin
Analyses > ANOVA > MANOVA

// Options
1. Variables dépendantes : Y1, Y2, Y3
2. Facteurs fixes : Groupe
3. Modèle multivarié : Trace de Pillai ou Wilks' Lambda

Analyse de Contrastes

Pourquoi ? Le "Sniper" statistique. Tester une hypothèse très précise.
💡 Exemple : "Je parie que le Groupe A est supérieur à la moyenne des Groupes B et C."
Pourquoi lui ? L'ANOVA dit juste "C'est différent". Les contrastes disent "C'est différent EXACTEMENT comme je l'avais prédit". 

# Définir les poids (Somme = 0)
# Ex: A(+2) vs B(-1) et C(-1)
contrasts(df$Groupe) <- c(2, -1, -1)

# Lancer l'ANOVA avec ces contrastes
model <- aov(Y ~ Groupe, data=df)
summary.aov(model, split=list(Groupe=list("MonHypothese"=1)))
# Python (via t_test sur OLS)
contrast = [0, 2, -1, -1] # À adapter selon l'ordre des coefs
t_test = model.t_test(contrast)
print(t_test)
// Chemin
Analyses > ANOVA > ANOVA

// Options
1. Ouvrir le menu "Contrastes"
2. Choisir le type (ex: Polynomial ou Helmert)
3. Ou utiliser le module optionnel "Rj" pour coder des contrastes personnalisés.

⚖️ Test T de Student

C'est quoi ? Une comparaison de moyennes entre 2 groupes.
💡 Exemple : "Les femmes gagnent-elles moins que les hommes ?"
Pourquoi lui ? Tes groupes sont assez grands et normaux. La moyenne est donc un bon résumé fiable à comparer.

📜 Les 3 Postulats (Conditions) :
  • ✅ Indépendance : Tes observations ne s'influencent pas entre elles.
  • ✅ Normalité : Tes données suivent une courbe en cloche (Loi Normale).
  • ✅ Homoscédasticité : Tes groupes ont des variances (dispersions) égales.
# Test T indépendant
t.test(Salaire ~ Sexe, data=data)
from scipy.stats import ttest_ind

groupA = df[df['Sexe'] == 'H']['Salaire']
groupB = df[df['Sexe'] == 'F']['Salaire']
t_stat, p_val = ttest_ind(groupA, groupB)
print(f"P-value: {p_val}")
// Chemin
Analyses > Tests t > Échantillons indépendants

// Options
1. Variable dépendante : Salaire (variable continue)
2. Variable de groupement : Sexe (2 groupes)
3. Tests d'hypothèses : T de Student (coché par défaut)

📊 ANOVA

C'est quoi ? Une comparaison de moyennes pour 3 groupes ou plus.
💡 Exemple : "Le salaire change-t-il selon la région (Nord/Sud/Est/Ouest) ?"
Pourquoi lui ? Faire 10 tests T de suite multiplierait le risque d'erreur. L'ANOVA teste tout d'un coup. 

# ANOVA
modele <- aov(Salaire ~ Region, data=data)
summary(modele)
from scipy.stats import f_oneway

# Séparer les groupes
g1 = df[df['Region'] == 'Nord']['Salaire']
g2 = df[df['Region'] == 'Sud']['Salaire']
g3 = df[df['Region'] == 'Est']['Salaire']

f_stat, p_val = f_oneway(g1, g2, g3)
print(f"P-value: {p_val}")
// Chemin
Analyses > ANOVA > ANOVA

// Options
1. Variable dépendante : Salaire
2. Facteurs fixes : Région (Groupes)
3. Post-tests (si significatif) : Tukey

📈 Corrélation (Pearson)

C'est quoi ? La mesure d'un lien linéaire entre 2 chiffres.
💡 Exemple : "Plus on est âgé, plus on est riche ?"
Pourquoi lui ? Tes données sont normales. Pearson est le standard pour voir si les points forment une ligne droite.

📉 Corrélation de Spearman

C'est quoi ? Une corrélation basée sur les rangs (classement).
💡 Exemple : "Le classement au concours est-il lié au QI ?"
Pourquoi lui ? Tes données ne sont pas des mesures parfaites. Spearman contourne le problème en transformant tout en classement. 

# Pearson (Classique)
cor.test(data$Age, data$Revenu, method="pearson")

# Spearman (Rangs)
cor.test(data$Satisfaction, data$Revenu, method="spearman")
from scipy.stats import pearsonr, spearmanr

# Pearson
print(pearsonr(df['Age'], df['Revenu']))

# Spearman
print(spearmanr(df['Satisfaction'], df['Revenu']))
// Chemin
Analyses > Régression > Matrice de corrélations

// Options
1. Déplacer les 2 variables (Age, Revenu) dans la boite.
2. Cocher "Pearson" (Lien linéaire) ou "Spearman" (Classement).
3. Cocher "Rapport de significativité" pour avoir la p-value.

U Test de Mann-Whitney

C'est quoi ? L'alternative tout-terrain au Test T.
💡 Exemple : "Comparer le niveau de stress (1-10) entre deux petits groupes."
Pourquoi lui ? Avec des petits groupes ou des notes subjectives, la moyenne ment. Ce test compare qui est "devant" l'autre en général.

H Test de Kruskal-Wallis

C'est quoi ? L'alternative tout-terrain à l'ANOVA.
💡 Exemple : "Comparer la satisfaction (1-5) de 3 hôtels."
Pourquoi lui ? Il classe tous les clients du plus au moins satisfait et regarde si un hôtel est toujours en tête de classement. 

# Mann-Whitney (2 groupes)
wilcox.test(Score ~ Groupe, data=data)

# Kruskal-Wallis (3+ groupes)
kruskal.test(Score ~ Groupe, data=data)
from scipy.stats import mannwhitneyu, kruskal

# Mann-Whitney
u_stat, p_val = mannwhitneyu(group1, group2)

# Kruskal-Wallis
h_stat, p_val = kruskal(group1, group2, group3)
